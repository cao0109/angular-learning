### 2. Angular 环境搭建

#### 2.1. 安装 Node.js

```bash
# 查看 node 版本
node -v
# 查看 npm 版本
npm -v
```

#### 2.2. 安装 Angular CLI

```bash
npm install -g @angular/cli
```

#### 2.3. 创建 Angular 项目

```bash
ng new [project-name] --style=scss --routing
```

#### 2.4. 启动 Angular 项目

```bash
ng serve --open
```
